# world-happiness-data-analysis
Exploratory Data Analysis of the World Happiness dataset using Python, SQL, and data visualization.

## Overview
This project explores the World Happiness dataset to understand factors influencing happiness levels across countries.

## Tools Used
- Python
- Pandas
- SQL
- Matplotlib
- Seaborn

## Key Questions
- Which countries are the happiest?
- How does GDP relate to happiness?
- What factors correlate most with happiness?

## Key Visualizations
- Correlation heatmap
- Top 10 happiest countries

## Dataset
World Happiness Report dataset.
